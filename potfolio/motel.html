Project Description
เป็นรับบจัดการข้อมูลหอพักโดยสามารถจัดการข้อมูลที่จำเป็นในการจัดการห้องพัก มีระบบจองห้องพัก การเซ็คอิน การเซ็คเอ้า ห้องพัก มีการออกรายงานของแต่ละเดือน มีระบบแจ้งเกือนกรณีไม่จ่ายค่าห้องพัก ตัวนี่ได้ทีมงานพัฒนาช่วยกัน 3 คน ช่วงหารายได้ระหว่างเรียน
