Project Description
เป็นระบบการขายสินค้าโดยมีระบบการทดลองส่วมใส่สินค้าเพื่อให้การขายสินค้ามีความน่าสนใจมากขึ้น ภายในเว็บไซต์มีระบบให้คะแนนสินค้า มีการนำ facebook api เข้ามาเกี่ยวข้องกับงาน มีระบบจัดการสินค้า โปรโมชั่นและระบบจัดการข้อมูลพื้นฐานของการขายสินค้า
